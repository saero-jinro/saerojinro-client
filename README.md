# ittime

<a alt="ittime" href="https://ittime.site">ittime</a>




## 프로젝트 소개
> 시간표 기반의 일정 관리 기능을 중심으로, 대규모 컨퍼런스 참가자와 운영진 모두가 효율적으로 일정을 확인하고 관리할 수 있는 웹 서비스를 개발하는 것이 본 프로젝트의 목표입니다.
